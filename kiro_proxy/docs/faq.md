# 常见问题

## 429 错误怎么办？
请求太频繁触发限制。代理会自动：
- 冷却该账号 5 分钟
- 切换到其他账号
- 冷却后自动恢复

## 对话太长报错？
Kiro 有长度限制。解决方法：
1. Claude Code 输入 `/clear` 清空对话
2. 告诉 AI 你之前在做什么

## Token 过期？
代理自动刷新。如果失败：
1. 点击「刷新 Token」
2. 或重新登录

## 如何添加多账号？
用不同账号多次登录，每次点击「扫描 Token」添加。

## 支持哪些模型？
- claude-sonnet-4（推荐）
- claude-sonnet-4.5（更强）
- claude-haiku-4.5（快速）
- claude-opus-4.5（最强）
