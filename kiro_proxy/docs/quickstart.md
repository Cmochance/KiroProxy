# 快速开始

## 运行代理
```bash
python run.py
```
启动后访问 http://localhost:8080

## 获取账号

### 在线登录（推荐）
1. 点击「账号」→「在线登录」
2. 选择 Google / GitHub / AWS
3. 浏览器完成授权，账号自动添加

### 扫描 Token
1. 先用 Kiro IDE 登录
2. 点击「扫描 Token」添加账号

## 配置客户端

### Claude Code
- Base URL: `http://localhost:8080`
- API Key: `any`
- 模型: `claude-sonnet-4`

### Codex CLI
- Endpoint: `http://localhost:8080/v1`
- API Key: `any`
- 模型: `gpt-4o`
